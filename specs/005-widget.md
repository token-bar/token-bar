# 005 — Widget

## Goal

Provide optional Notification Center and Desktop widgets.

---

## Display

* Provider
* Usage %
* Progress bar
* Reset date

---

## Constraints

Must tolerate infrequent WidgetKit refreshes.

---

## Acceptance Criteria

Widget remains functional independently from menu bar.

---

## Test Cases

* no provider configured
* stale data
* refresh failure
